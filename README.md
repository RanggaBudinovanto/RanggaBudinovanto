### 🧑‍💻 About Me

Undergraduate **Informatics student at the Faculty of Computer Science, UPN Veteran Jakarta** (6th semester), with a core focus on **web development**, **UI/UX design**, and **cybersecurity**.

Currently working on my **thesis** on security vulnerability analysis of the SIMPEL UPNVJ web application, using **NIST SP 800-115**, **OWASP Top 10 2025**, and **CVSS v4.0** methodologies — with tools such as Burp Suite, ZAP, SQLMap, Nmap, and WhatWeb on Kali Linux.

Outside academics, I enjoy building personal projects ranging from web apps and multiplayer games to non-generic, production-ready UI systems.
